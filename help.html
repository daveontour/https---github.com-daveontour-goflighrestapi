/getFlights

Parameters:

Airport - Three letter IATA airport code to the desired airport
al or airline - Two letter IATA code for the airline, eg. BA, DL, LH, MH (default: all airlines)
d or direction - number of hours relative to "now" to start looking for allocations, eg. -3 (default: -12)
r or route - number of hours relative to "now" to stop looking for allocations, eg. 12 (default: 24)
from - number of hours relative to "now" to start looking for allocations, eg. -3 (default: -12)
to - number of hours relative to "now" to stop looking for allocations, eg. 12 (default: 24)

/getResources/[Airport]
Parameters:

Airport

Three letter IATA airport code to the desired airport

flt or flight

Flight Number, eg. QF001, EK23, BA007 (default: all flights)

al or airline

Two letter IATA code for the airline, eg. BA, DL, LH, MH (default: all airlines)

rt or resourceType

one of CheckIn, Gate, Stand, Carousel, Chute. (default: all types are returned)

id or resource

the name of the individual reource to query (default: all resources)

from

number of hours relative to "now" to start looking for allocations, eg. -3 (default: -12)

to

number of hours relative to "now" to stop looking for allocations, eg. 12 (default: 24)

updatedSince

return records that have been updated from the date, e.g. 2023-07-16T13:00:00

{custom field name}

 



-

Examples:

Find the flights allocated to checkin desk 100 from now until 12 hours from now
/getResources/APT?from=0&to=12&rt=CheckIn&id=100

Find all the resources allocated to flight QF100
/getResources/APT?flt=QF100

Find all the resources allocated to Emirates (EK)
/getResources/APT?al=EK

Find all the resources allocated to British Airways (BA) for the next 3 days
/getResources/APT?al=BA&from=0&to=72

Find all the resources where the custom field name Sh--_GroundHandler of the assigned flight is EAS
/getResources/APT?Sh--_GroundHandler=EAS

/getConfiguredResources/[Airport]
Returns all the configured resources for the specified Airport

Airport - Three letter IATA airport code to the desired airport

/getConfiguredResources/[Airport]/{ResourceType}
Returns all the configured resources of the specified type for the specified Airport

Airport - Three letter IATA airport code to the desired airport
